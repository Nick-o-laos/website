# How to Add Your Files

This folder contains the downloadable files for your portfolio projects.

## File Structure

Place your files here with the following naming convention:

### Project Files Mapping

1. **Machine Learning in Finance**
   - `ML_Finance_Report.pdf` - Your PDF report
   - `ML_Finance.ipynb` - Your Jupyter notebook

2. **Bitcoin Price Analysis**
   - `Bitcoin_Analysis.Rmd` - Your R Markdown file
   - `Bitcoin_Report.pdf` - Your PDF report

3. **Loan Programs Analysis**
   - `Loan_Analysis.ipynb` - Your Jupyter notebook
   - `Loan_Report.pdf` - Your PDF report

4. **Supply Chain Optimization**
   - `Supply_Chain.ipynb` - Your Jupyter notebook

5. **Football Analytics**
   - `Football_Analytics.ipynb` - Your Jupyter notebook
   - `Football_Report.pdf` - Your PDF report

6. **Survival Analysis**
   - `Survival_Analysis.Rmd` - Your R Markdown file
   - `Survival_Report.pdf` - Your PDF report

7. **Car Rentals Data Analysis**
   - `Car_Rentals.ipynb` - Your Jupyter notebook

8. **Financial Analysis on USD Funds**
   - `USD_Funds.ipynb` - Your Jupyter notebook
   - `USD_Funds_Report.pdf` - Your PDF report

9. **PMT & Riemann Hypothesis**
   - `PMT_Riemann.py` - Your Python script
   - `PMT_Riemann_Report.pdf` - Your PDF report

## How to Add Files

1. Copy your files to this `/files/` folder
2. Make sure the filenames match exactly as listed above
3. The download buttons will automatically work

## Supported File Types

- `.pdf` - PDF Reports
- `.ipynb` - Jupyter Notebooks
- `.Rmd` - R Markdown files
- `.py` - Python scripts
- `.r` - R scripts

## Note

Files in this folder will be publicly accessible. Make sure you only include files you want to share.
